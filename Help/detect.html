Keyboard Shortcuts

Key Action Description
Ctrl+O Open Open a Split Image (.tif, .tiff). You can open multiple images at once.
Ctrl+D Detect Display Cone Detection dialog. On this dialog you can adjust detection parameters, then chose to detect cones either on the current image, or on all open images.
Ctrl+S Save Save detection results of the current image.
Ctrl+A Save All Save detection results of all open images.
Ctrl+B Adjust Switch to “Ajust” mouse mode. In this mode, you can adjust brightness and contrast of the image by dragging the mouse.
Ctrl+M Mark Switch to “Mark Cones” mouse mode. In this mode, clicking on the image will add a cone glyph to the detection results.
Ctrl+E Erase Switch to “Erase Cones” mouse mode. In this mode, clicking on a cone glyph will remove it from the detection results.
Ctrl+Z Undo Undo the last “Mark” or “Erase” operation.
F1 Help Display this help screen.
