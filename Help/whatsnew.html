What's new – Cone Detection Utility

Version 1.1.003 (2022-03-11)

  • New tool button: Toggle Voronoi diagram display.

Version 1.1.002 (2022-03-06)

  • Fixed bug with dragging marker outside of the image.
  • Brightness/Contrast changes can be undone with the “Undo” button.
  • Minimal distance between markers, which used to be always equal to 3 pixels, is now tied to the Gliph Size. To reduce the distance, simply reduce the Gliph Size.

Version 1.1.001 (2022-02-13)

  • Default mouse behavior (changing image brightness/contrast) disabled in all mouse modes except Adjust.
  • Brightness/contrast is now set separately for each image, and it is not carried over to other images.
  • A new mouse mode Move Cones has been added to the toolbar. In this mode you can click on a cone marker center and drag the marker to another location.
  • Fixed issues with deleting markers in Erase M mouse mode at the image borders.
  • Disabled default behavior of keystrokes such as “W”, “R”, “X”, “Y”, etc.
  • After dragging in image files from a directory, the default save directory is changed to the one where the files were dragged in from.

Version 1.1.0 (2021-05-17)

  • Change mouse cursor to OpenHand when Shift is pressed, and to MoveAll when scrolling the image with Shift+Left Button or Middle Button.
  • Prevent new markers from being added too close to an existing marker (≤ 3 pixels).
  • Up and Down arrows switch between open images (even if the image list does not have focus).
  • The “Detect” dialog displays a list of open images with check boxes, allowing the user to select which images to detect (and replace existing detections). Initially only images that don't have any markers on them are checked, but there is a “√” header to check/uncheck all.
  • A “√” symbol is placed next to image name in the list of open images if there is a matching *.csv file in the same directory as the image (*.tif) file. An image may actually have some markers on it, but they are not saved.
  • Fixed problem with long file names (Use Qt instead of STL for reading/writing text files).
  • Added Split → Delete All menu command for deleting all cone markers.
  • Added Erase M tool button to switch to mouse mode where drawing a contour with mouse erases all cone markers inside.
  • Added Split → Keyboard Shortcuts menu command, which displays a dialog box that allows users to customize keyboard shortcuts for most actions.
  • Toggle pixel interpolation for image zoom resampling (Ctrl+I or menu Split → Toggle Interpolation).
  • F2 toggles cone glyph visibility.
  • Fixed Unicode issues under Windows.
